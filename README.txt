Paket Website Bisnis Kecil
=========================
Isi paket:
- index.html
- about.html
- services.html
- testimonials.html
- contact.html

Instruksi cepat:
1. Ekstrak ZIP.
2. Buka index.html di browser (langsung bisa, Tailwind via CDN).
3. Untuk mengganti teks/gambar, edit file .html dengan editor teks.
4. Untuk hosting, upload semua file ke hosting statis (GitHub Pages, Netlify, Vercel, dsb.)

Catatan:
- Formulir kontak di contact.html hanya demo (tidak mengirim data).
- Ganti placeholder (Nama Bisnis, alamat, nomor telepon) sesuai bisnis Anda.
